Relaxed Minimum Spanning Tree 
=============================

Python implementation of the Relaxed Minimum Spanning Tree  (RMST) algorithm. 

To install 
```
pip install .
```

To use, 
```
from RMST import RMST

G_RMST = RMST(G)

```
for a networks graph `G`. 

See the `test` folder for more details.
